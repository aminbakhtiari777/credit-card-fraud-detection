#Credit Card Fraud Detection

This machine learning project focuses on detecting fraudulent credit card transactions using a highly imbalanced real-world dataset.

## Project Goal

The goal is to identify fraudulent transactions while minimizing false alarms and missed fraud cases.

## What I Practiced

- Loading and exploring real financial data
- Working with imbalanced datasets
- Train/Test split
- Random Forest Classification
- Fraud detection metrics
- Confusion Matrix
- Precision, Recall and F1-Score
- Model evaluation
- Saving trained models

## Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Model

- Random Forest Classifier

## Results

- Accuracy: ~99.95%
- Precision (Fraud): ~97%
- Recall (Fraud): ~77%

This project demonstrates why accuracy alone is not enough for fraud detection problems.
